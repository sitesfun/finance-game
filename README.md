# 💸 Money Quest

Money Quest — це проста браузерна гра з фінансової грамотності, створена на HTML, CSS та JavaScript.

Гравець ставить собі фінансову ціль (мрію), заробляє гроші, відкладає їх на заощадження, стикається з випадковими подіями та намагається накопичити необхідну суму для досягнення своєї мети.

## 🎮 Можливості

* Заробіток випадкової суми грошей
* Система заощаджень
* Випадкові події, які можуть допомогти або завадити накопиченню
* Прогресбар досягнення цілі
* Простий та зрозумілий інтерфейс
* Гейміфікація процесу накопичення коштів

## 🎯 Мета гри

Навчити основам фінансової грамотності через гру:

* планування бюджету;
* накопичення коштів;
* управління доходами та витратами;
* прийняття фінансових рішень.

## 🛠️ Технології

* HTML5
* CSS3
* JavaScript (Vanilla JS)
